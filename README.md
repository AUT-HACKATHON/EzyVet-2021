## Local Instance

```sh
git clone https://github.com/AUT-HACKATHON/EzyVet-2021.git ./ezyvet-2021
```

and then

```
cd ./ezyvet-2021 && yarn;
cd ./frontend && yarn
cd ..
cp .env.example .env
```

then fill out the .env providing your mongodb information and jwt secret

```
npm run dev
```
